# 角色境界设计

本文用于定义角色境界、底层 `realmLv`、品阶映射，以及凡武到修仙、再到飞升后的完整递进。

结构化配置同步输出到：

- `packages/server/data/content/realm-levels.json`

## 1. 总原则

- 角色底层只使用一个连续等级：`realmLv`
- 境界名、境界阶段、所属路径、对应品阶，全部由 `realmLv` 推导
- `12 lv = 1 品阶段`
- 凡武期允许一境一级，用来做细密的成长和叙事递进
- 修仙期采用规则段：`6 lv = 1 修仙大境`
- 飞升后不再沿用固定 `6 lv = 1 境`，改为高阶散段境界，每一级都是独立关口

## 2. 三段结构

整个体系分为三段：

- `realmLv 1-18`：凡武期
  - 一境一级
  - 先练武，再入玄，最后抵达仙门前夜
- `realmLv 19-72`：修仙期
  - `6 lv = 1 大境`
  - 从练气一路走到渡劫
- `realmLv 73-96`：飞升后期
  - 不再按固定 6 级一境
  - 先从仙位往上爬，再入圣，最后入帝

## 3. 品阶映射

| 品阶 | realmLv | 主覆盖境界 |
| --- | --- | --- |
| 凡阶 | 1-12 | 凡胎到天玑 |

| 玄阶 | 25-36 | 筑基、金丹 |
| 地阶 | 37-48 | 元婴、化神 |

| 灵阶 | 61-72 | 大乘、渡劫 |

- `筑基 = 玄阶`
- 大多数品阶会覆盖两个修仙大境
- 飞升后不再机械按“两境一品阶”推进，而是按世界观单独命名

## 4. 凡武期 `realmLv 1-18`

- 开场先有明确练武味道，不要一上来就太玄
- `北斗七境` 保留，但放在武道中段
- 末段要有明确的江湖巅峰感
- `叩仙门` 作为凡武尽头与练气前夜保留

## 5. 修仙期 `realmLv 19-72`

从 `realmLv 19` 开始正式踏入修仙序列。

修仙期统一使用 6 段阶段：

| 段位偏移 | 阶段名 |
| --- | --- |
| `+0` | 前期 |
| `+1` | 中期 |
| `+2` | 后期 |
| `+3` | 巅峰 |
| `+4` | 大圆满 |
| `+5` | 半步下境 |

例如：

- `realmLv 19` = 练气前期
- `realmLv 23` = 练气大圆满
- `realmLv 24` = 半步筑基

## 6. 飞升后期 `realmLv 73-96`

飞升后不再延续修仙期那种固定 6 级一境。

这里的设计原则是：

- `73-80`：从飞升踏入真正仙位
- `81-84`：从仙走到圣
- `85-96`：帝道十二关，后半段不再执着于每级都带一个“帝”字
