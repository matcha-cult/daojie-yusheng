# 角色境界设计

本文用于定义角色境界、底层 `realmLv`、品阶映射，以及凡武到修仙、再到飞升后的完整递进。

结构化配置同步输出到：

- `packages/server/data/content/realm-levels.json`

## 1. 总原则

- 角色底层只使用一个连续等级：`realmLv`
- 境界名、境界阶段、所属路径、对应品阶，全部由 `realmLv` 推导
- `12 lv = 1 品阶段`
- 凡武期允许一境一级，用来做细密的成长和叙事递进
- 修仙期采用规则段：`6 lv = 1 修仙大境`
- 飞升后不再沿用固定 `6 lv = 1 境`，改为高阶散段境界，每一级都是独立关口

## 2. 三段结构

整个体系分为三段：

- `realmLv 1-18`：凡武期
  - 一境一级
  - 先练武，再入玄，最后抵达仙门前夜
- `realmLv 19-72`：修仙期
  - `6 lv = 1 大境`
  - 从练气一路走到渡劫
- `realmLv 73-96`：飞升后期
  - 不再按固定 6 级一境
  - 先从仙位往上爬，再入圣，最后入帝

## 4. 凡武期 `realmLv 1-18`

设计目标：

- 开场先有明确练武味道，不要一上来就太玄
- `北斗七境` 保留，但放在武道中段
- 末段要有明确的江湖巅峰感
- `叩仙门` 作为凡武尽头与练气前夜保留

## 5. 修仙期 `realmLv 19-72`

从 `realmLv 19` 开始正式踏入修仙序列。

修仙期统一使用 6 段阶段：

| 段位偏移 | 阶段名 |
| --- | --- |
| `+0` | 前期 |
| `+1` | 中期 |
| `+2` | 后期 |
| `+3` | 巅峰 |
| `+4` | 大圆满 |
| `+5` | 半步下境 |

例如：

- `realmLv 19` = 练气前期
- `realmLv 23` = 练气大圆满
- `realmLv 24` = 半步筑基

## 7. 当前突破要求

### 7.1 规则约束

- 当前版本前期突破不再是纯经验到顶即自动上升，而是 `境界经验圆满 + 突破要求全部满足` 后，玩家手动确认突破。
- 突破要求当前支持三类：`item`、`attribute`、`technique`。
- 其中 `item` 类要求在突破成功后会被直接消耗，前端弹层必须明确提示“材料会被消耗”。
- 若某条突破要求未显式配置 `hidden: true`，则默认直接展示，不走 `???` 隐藏。
- 武道段突破现在按“外练筋骨 -> 易筋洗髓 -> 养气通脉 -> 定势成宗师”的思路设计。
- 当前模型中，`attribute` 是基础门槛；带 `reduceAttrRequirementPct` 的 `item` 和 `technique` 不是硬门槛，而是可选减免项。
- 减免项统一按乘算叠加，不做加法叠加。
- 示例：一组材料减免 `20%`，另一组功法减免 `50%`，最终属性要求乘区为 `0.8 × 0.5 = 0.4`，也就是总共降到原要求的 `40%`。
- 若连续两组材料都各减免 `20%`，则最终乘区为 `0.8 × 0.8 = 0.64`，等价于总要求降低 `36%`。
- 材料只作为药引、淬体外物或冲关辅材；即使不收集材料，只要基础属性足够高，也允许硬冲突破。
- 功法减免幅度显著高于材料减免，用于体现“修行火候”比外物更重要。
- 当前已实装并定稿的是 `realmLv 1 -> 19` 的武道突破要求，以及 `24 -> 25` 的筑基前要求。

### 7.4 设计意图

- 前 `1-8` 级以外练筋骨、易筋洗髓、养气通脉为主，材料只作药引，核心门槛是体魄、神识、根骨、悟性与功法层数。
- `9-16` 级开始强调“功法火候”和“势”的完成度，因此加入黄阶功法大成、圆满、双功法大成等高减免项。
- 材料减免当前主走 `20%` 一档，作为中等幅度辅助；功法减免主走 `60%-80%`，显著高于材料，体现“修行火候重于外物”。
- 矿洞、遗迹、兽谷材料只在中后段作为冲关辅材使用，避免武道段完全异化成刷素材游戏。
- `18 -> 19` 的“破凡入真”也改成可自由取舍的模型：不刷材料、也不靠功法减免时，仍可用高属性硬冲；若准备齐全，则要求会被大幅压低。
